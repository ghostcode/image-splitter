# 图片分割工具

一个现代化、高性能的网页图片分割工具。支持自定义网格布局、实时预览和批量导出。所有处理均在浏览器本地完成，无需上传服务器，保护您的隐私。

---

## 功能特性

- **直观的操作界面**
  - 支持拖拽上传或点击选择图片文件
  - 支持 JPG、PNG、WebP 等常见格式

- **实时网格预览**
  - 调整分割参数时即时预览效果
  - 网格自动保持原图宽高比，预览无变形
  - 交错动画展示网格生成过程

- **灵活的分割设置**
  - 自定义行数和列数（1-20）
  - 内置 8 种快捷预设（如 2×2、3×3、九宫格等）
  - 实时显示切片尺寸和总块数

- **多样化的导出方式**
  - **单独下载**：点击任意切片直接保存为 PNG
  - **ZIP 打包**：一键将所有切片打包为 ZIP 文件下载
  - 包含实时打包进度条和成功提示

- **隐私安全**
  - 所有图片处理均在本地 Canvas 完成
  - 数据不上传云端，无需担心隐私泄露

- **精致的视觉体验**
  - 深色科技风界面设计
  - 流畅的过渡动画
  - 响应式布局，适配桌面与移动端

## 快速开始

无需安装依赖，直接在浏览器中打开 HTML 文件即可使用。

```bash
# 1. 下载项目文件
# 2. 直接用浏览器打开 index.html
```

### 使用步骤

1. **上传图片** - 拖拽图片到上传区域，或点击选择文件
2. **设置网格** - 调整行列数或选择快捷预设
3. **预览效果** - 在右侧预览区查看分割效果
4. **执行分割** - 点击「开始分割」按钮
5. **下载切片** - 单击切片单独下载，或点击 ZIP 按钮批量导出

## 核心逻辑

### 图片分割原理

利用 Canvas 的 `drawImage()` API 进行图片裁剪：

```javascript
// 计算每个切片的尺寸
const pieceWidth = imageWidth / cols;
const pieceHeight = imageHeight / rows;

// 循环裁剪
for (let r = 0; r < rows; r++) {
    for (let c = 0; c < cols; c++) {
        ctx.drawImage(
            image,
            c * pieceWidth, r * pieceHeight,  // 源图裁剪起点
            pieceWidth, pieceHeight,           // 源图裁剪尺寸
            0, 0,                              // 画布绘制起点
            pieceWidth, pieceHeight            // 画布绘制尺寸
        );
    }
}
```

### ZIP 打包流程

使用 JSZip 库将切片打包：

```javascript
const zip = new JSZip();
const folder = zip.folder('images');

// 添加文件
folder.file('slice_1.png', base64Data, { base64: true });

// 生成并下载
const content = await zip.generateAsync({ type: 'blob' });
```
